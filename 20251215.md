https://leetcode.cn/problems/move-zeroes/?envType=problem-list-v2&envId=array

```java
class Solution {
    public void moveZeroes(int[] nums) {

        if(nums.length <= 1){
            return;
        }

        int i,j;

        for(i =0 ;i < nums.length;i++){
            if( nums[i] == 0 )
                break;
        }

        for(j=i,i=i+1; i < nums.length;i++){
            if(!(nums[i] == 0)){
                nums[j] = nums[i];
                nums[i] = 0;
                j++;
            }
        }
    }
}
```
